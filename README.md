# IP Camera Live Stream using OpenCV

This project demonstrates how to stream live video from an IP camera using Python and OpenCV.

##  Features
- Live IP camera video streaming
- FPS (Frames Per Second) display
- Screenshot capture using keyboard
- Error handling for connection issues

##  Requirements
- Python 3.x
- OpenCV

##  Install dependencies:
- pip install opencv-python

##  How to Use
1. Connect your mobile/IP camera to the same network.
2. Get the camera stream URL (e.g. from IP Webcam app).
3. Replace the URL in the script:
- url = ```"http://<IP>:<PORT>/video"```

##  Controls
- `q` → Quit application
- `s` → Save screenshot

##  Applications
- CCTV systems
- Surveillance projects
- Computer vision preprocessing
- Real-time video analysis
